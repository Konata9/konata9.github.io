# 0. 什么是 DOM ？

DOM 全称 Document Object Model，即文档对象模型，是前端开发中最重要的概念之一。网页由一个个 HTML 标签组成，而 DOM 则提供了访问和操作 HTML 的标准方法。利用 DOM 提供的 API，我们就能使用 JavaScript 对页面进行各种操作。
